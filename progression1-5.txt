Week 1 – Filter หาสินค้า (พื้นฐาน)

    รายละเอียด |

    สร้างฟังก์ชัน filter เพื่อค้นหาสินค้าใน array

    เป็นพื้นฐานสำหรับใช้งานจริงในอนาคต

    Reflect |

    ได้ฝึกใช้ฟังก์ชัน filter กับข้อมูล

    เข้าใจการค้นหาข้อมูลขั้นพื้นฐานใน JavaScript

Week 2 – DOM + Filter บนหน้าเว็บ

    รายละเอียด |

    ใช้ DOM เข้าถึง element ของ HTML ด้วย JavaScript

    เพิ่ม input ช่องค้นหา และใช้ filter ทำงานจริงกับ data

    อัปเดตผลลัพธ์ในหน้าเว็บทันทีเมื่อพิมพ์

    Reflect |

    ได้เรียนรู้วิธีใช้ DOM และ event listener

    ได้ฝึกนำฟังก์ชัน filter มาใช้งานจริงในเว็บ

Week 3 – Fetch ข้อมูลจาก API

    รายละเอียด |

    เปลี่ยนจาก mock data แบบ array ธรรมดาไปใช้ API แทน

    ใช้ fetch เพื่อดึงข้อมูลจาก mock API (เช่น Mocki, Mocky)

    ทำให้เหมือนการทำงานกับ backend จริงมากขึ้น

    Reflect |

    ได้เรียนรู้วิธีใช้ fetch และ async/await

    เข้าใจการทำงานกับข้อมูลจาก API

Week 4 – เพิ่มการ Sort ข้อมูล

    รายละเอียด |

    เพิ่ม checkbox เพื่อเรียงสินค้าตามชื่อ (sort by name)

    ใช้ event listener ตัวใหม่ (เช่น change event)

    แสดงผลลัพธ์เรียงลำดับตามที่เลือก

    Reflect |

    ได้เรียนรู้การใช้ sort กับข้อมูลใน array

    เข้าใจการจัดการ event ที่หลากหลายมากขึ้น

Week 5 – ขยายการ Sort (ตามราคา)

    รายละเอียด |

    เพิ่ม checkbox อีกตัวสำหรับเรียงสินค้าตามราคา

    ใช้เงื่อนไขและฟังก์ชัน sort เพื่อสลับการเรียงมากไปหาน้อย และน้อยไปหามาก

    Reflect |

    ได้ทบทวนและฝึกใช้ sort อีกครั้ง แต่กับข้อมูลที่ต่างออกไป (numeric)

    เห็นภาพรวมว่าฟีเจอร์เล็ก ๆ สามารถค่อย ๆ สร้างให้เว็บสมบูรณ์ขึ้นทีละขั้นได้

สิ่งที่ได้จากทั้งหมด (Week 1–5)

    พื้นฐานการทำงานกับข้อมูล: เริ่มจาก filter ธรรมดา → ค้นหาข้อมูลจริงบนเว็บ

    การจัดการ DOM และ Event: เรียนรู้การเชื่อม interaction ของผู้ใช้กับข้อมูล

    การดึงข้อมูลจาก API: เข้าใจวิธี fetch และทำงานกับ asynchronous data

    การจัดการและแสดงผลข้อมูล: เพิ่มการ sort ทั้งแบบ text และ numeric เพื่อปรับรูปแบบการแสดงผล

    การสร้างเว็บที่พัฒนาเป็นขั้นตอน: เห็นความก้าวหน้าทีละสัปดาห์จาก logic พื้นฐาน จนถึงการทำงานใกล้เคียงเว็บจริง
